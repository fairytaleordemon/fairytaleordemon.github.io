引言

Nginx 是一款轻量级的 http 服务器，可以实现负载均衡和反向代理。
负载均衡：通俗的来讲就是电影院有多个厅，一个厅当然不够咯。所以需要把顾客安排到不同的厅。 Nginx 就是把请求（人）拿过来，然后检查看看那个服务器（厅）还可以容纳（人），就把请求（人）给它。
反向代理：典型的来说，就是你找你去找女朋友。但是找不到呀，就需要通过媒婆详情啊。Nginx 就是把来的所有请求（需要找女朋友的人），转发给对应的服务器（需要找男朋友的人）。

使用

下载 Nginx

Nginx 下载地址
进入下载地址，选择自己需要的版本，我这里选择的是 1.8.0。如果是 window 用户，请下载windows版本的安装包。

安装 Nginx

先安装依赖组件，这些组件很多应用都会用得到

yum -y install zlib zlib-devel openssl openssl-devel pcre pcre-devel

Mac 以及linux 用户使用以下命令解压。（有解压工具的无视）。

1.tar -xvf nginx-1.8.0.tar.gz
2.cd nginx-1.8.0.tar.gz
3../configure 　　　　　　#编译
4.make && make install　#安装

启动 Nginx

1.cd sbin
2.nginx -t　　　　　　　　#检查配置文件中是否有语法错误
3.nginx　　　　　　　　　#启动 Nginx

其他 Nginx 启动常用命令
nginx -s reload 　＃重启当前服务
